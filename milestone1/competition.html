Here are my thoughts about this website right here: asana


Here are the positives sides of this website

  • This website has some good contrast at the start. Having some colors with it too.
  • This has some good basic navigation bar. With the logo heading back to the main website
  • Has a globe button to change language of the website without using the help with translating.
  • Has videos and animations that gives out to the website.

Here are the negative sides of this website.

  • There's a banner on the top of the page that doesn't seem like it fits in well.
  • There's an AI chat bot that takes up 1/4 the page. Making the user to click the X button.
  • There's a scroll bar on the bottom of the page that serves little to no purpose.

Here are some changes that I wanted to do.

  • Adjust the page so there isn't a scroll bar at the bottom
  • Having the AI bot to be smaller or just to be an icon.
  • having the banner below the navigation so everything stays the same spot.
